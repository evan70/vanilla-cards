Getting Started

Quick Start

Include the Vanilla Cards bootstrap in your HTML:

<link rel="stylesheet" href="vanilla-cards.css">
<script type="module" src="vanilla-cards.js"></script>

Usage

Use the card component in your HTML:

<vanilla-card variant="elevated">
    <h3 class="card__title">Card Title</h3>
    <p class="card__description">Card content goes here.</p>
</vanilla-card>

Components

<vanilla-card>

The main card component with multiple variants and configurations.

Attributes

Attribute Type Default Description

Styling

CSS Custom Properties

Customize cards using CSS custom properties:

:root {
    --card-padding: 1.5rem;
    --card-radius: 1rem;
    --card-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
    --brand-gold: #d4af37;
}

BEM Classes

Use BEM naming for card content:

.card__title { }
.card__description { }
.card__footer { }
.card__image { }
.card__action { }

Accessibility

Built-in Accessibility

  • ARIA roles and attributes
  • Keyboard navigation support
  • Focus management
  • Screen reader compatible

Clickable Cards

For keyboard accessibility, use tabindex:

<vanilla-card clickable tabindex="0">
    Click me via keyboard
</vanilla-card>

API Reference

JavaScript API

const card = document.querySelector('vanilla-card');

// Get/set variant
card.variant = 'filled';

// Get/set clickable
card.clickable = true;

// Get/set disabled
card.disabled = false;

// Get/set href
card.href = '/page';

// Focus the card
card.focus();

// Blur the card
card.blur();

Events

Event Description
clickFired on card click
focusFired on focus
blurFired on blur
